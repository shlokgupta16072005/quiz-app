# Quiz App

## Description
A web-based quiz application that allows users to answer multiple-choice questions and see their score at the end.

## Features
- Multiple-choice questions
- Score calculation
- User-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript

## Installation
# Clone the repository
git clone https://github.com/yourusername/quiz-app.git

# Open the project
cd quiz-app
# Open index.html in your browser
